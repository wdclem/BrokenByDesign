# BrokenByDesign

Learn by fixing: clone a repo with intentional bugs, fix via PR, get AI review, merge.

## For learners: clone → solve → PR

1. **Clone** this repo (or fork, then clone your fork).
2. Pick a challenge (e.g. [Challenge 01 — broken frontend](challenges/01-broken-frontend/SPEC.md)), run the app (see the challenge’s `app/` README).
3. **Fix** the bugs. Work on a branch (e.g. `fix/01-broken-frontend`).
4. **Open a PR** with a description: what was broken, root cause, and what you changed. AI or maintainer will review; if it passes, merge.
