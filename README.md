# BrokenByDesign

Learn by fixing: fork the repo (main is protected), fix via PR on your fork, automated tests run, merge.

## For learners: fork → solve → PR on your fork

1. **Fork** this repo (upstream `main` is protected), then **clone your fork**.
2. Pick a challenge (e.g. [Challenge 01 — broken frontend](challenges/01-broken-frontend/SPEC.md)), run the app (see the challenge’s `app/` README).
3. **Fix** the bugs. Work on a branch (e.g. `fix/01-broken-frontend`).
4. **Open a PR on your fork** with a description: what was broken, root cause, and what you changed. Automated tests run on the PR; if they pass, merge.
